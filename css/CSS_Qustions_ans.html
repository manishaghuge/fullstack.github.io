7. What is Diff between border and outline?
  1. Border Allow to set each edges with different width or color. outlines won't allow us to set each edge to a different width, or set different colors and styles for each edge. An outline is the same on all sides.
  2. Border can be in rectangle shape , but Outlines may be non-rectangular.
  3. Border can change the size and position of the element , but outlines don't change the size or position of the element.
  4. Outlines do not take up space, because they always placed on top of the box of the element which may cause them to overlap other elements on the page.

10. What is z- index

The z-index property specifies the stack order of an element.An element with greater stack order is always in front of an element with a lower stack order.
z-index only works on positioned elements (position: absolute, position: relative, position: fixed, or position: sticky).
