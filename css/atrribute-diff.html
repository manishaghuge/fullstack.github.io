No. Atrribute Example discription
1. [attribute] [title]{ } This css is apply to all title attribute of element. its not depends on value of title attribute
2. [attribute="value"] [title = "seven"]{ } this css is apply on title attribute whose value is only "seven", if the value is "seven mentor" then not apply the css.
Allow values :- seven.
Not allow values : seven mentor,seven-mentor,sevenmentor, mentor seven
3. [attribute~="value"] [title ~= "seven"]{ } this css is apply on title attribute whose value is "seven" or "seven mentor" "its seven time" or "this no. seven" , but if the value is "tseven" then css not work..
Allow values :- seven, seven mentor,its seven time, its seven .
Not allow values : seven-mentor, mentor-seven
4. [attribute|="value"] [title |= "seven"]{ } this css is apply on title attribute whose value is "seven" or"seven-text",But if the value is "seven mentor" or "its-seven-time" or "tseven" or "t-seven" on this all example the css is not work.
Allow values :- seven, seven-mentor
Not allow values : seven_mentor, seven mentor, its-seven-time,t-seven", "tseven
5. [attribute^="value"] [title ^= "seven"]
Allow values :- seven, seven-mentor
Not allow values : seven_mentor, seven mentor its seven time its-seven-time,t-seven", "tseven
6. [attribute$="value"] [title $= "seven"] css apply when value find at end of string
Allow values :- mentorseven, mentor_seven, tring-seven, seven , string seven
Not allow values : Sevenstrnig, seven tring,(not accept )
7. [attribute*="value"] [title *= "seven"]
Allow values :- allow all values where seven is present
